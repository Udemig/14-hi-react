# Backend Kurulumu

1. .env dosyasını derss videosunun altından al ve projene aktar
2. .env içerisinden `NEXT_PUBLIC_API_URL` projenin çalıştığı porta göre ayarla
3. .env içerisinden `NEXT_PUBLIC_USER_ID` benzersiz kullanıcı id'ni yaz
4. `mongoose` ve `stripe` kütüphanelerini npm ile indir
5. `/src/app/api` klasörünü final projeden kopyalıp kendi projene aktar
