# useRef

- useRef, javascriptteki documentGetElementById, querySelector gibi methodların alternatifidir.
- useRef, react'ta şu durumlarda kullnılabilir.

1. DOM Elementlerine erişim sağlamak
2. Component re-render olmadan değer saklama

# Higher Order Components (HOC)

- Üst Düzey Bileşen

- Normal component'lardan farklı olarak `self-closing` değildir ve kendi kapanış etiketi vardır.

- Açılış / kapanış etiket içerisinde yer alan içerik compoent'a `children` propuyla aktarılır

- Bir component farklı bir component'ı/element'i kapsadığı için adı hoc'dur.
